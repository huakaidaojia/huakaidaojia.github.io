【npm】下包过程中报了while-parsing-near的错误
Created 2018-05-05| Updated 2020-03-25|npm
Post View:

在装包过程中如果报了这个错误，Unexpected end of JSON input while parsing near '...noattachment':false ，通常是包的缓存出问题了，只要是while parsing near就是包的缓存问题

解决方案

  1. 一般全局安装的包会存放到C:\Users\自己的用户名文件夹\AppData\Roaming\npm这个目录里
  2. 安装包的时候，默认会把包的缓存到 C:\Users\自己的用户名文件夹\AppData\Roaming\npm-cache 这个目录中
  3. 如果以后再 npm install 装包的时候报了while parsing near这个错误，删除npm-cache缓存目录
